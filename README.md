# 🩺 Explainable ML for Early Prediction of Hepatitis B

## 📌 Overview
This project focuses on building a Machine Learning system to predict Hepatitis B at an early stage using clinical and demographic data. It also incorporates Explainable AI techniques to make model predictions interpretable and trustworthy.

---

## 🚀 Features
- Early prediction of Hepatitis B using patient data
- Data preprocessing and feature engineering for improved model performance
- Exploratory Data Analysis (EDA) for understanding data patterns
- Multiple classification models built and optimized using Scikit-learn
- Hyperparameter tuning for better accuracy and generalization
- Explainable AI using SHAP to interpret model predictions
- Evaluation using accuracy, precision, recall, and F1-score

---

## 🧠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- SHAP (Explainable AI)
- Matplotlib / Seaborn (for visualization)

---

## 📊 Workflow
1. Data Collection (clinical + demographic dataset)
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Building (Classification algorithms)
6. Hyperparameter Tuning
7. Model Evaluation
8. Explainability using SHAP

---

## 📈 Model Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score

---

## 🔍 Explainable AI (SHAP)
SHAP was used to interpret model predictions and identify the most influential medical features affecting Hepatitis B prediction. This improves transparency and trust in the model.
